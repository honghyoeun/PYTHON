**Q1 집합(정보처리기사 20년 2회)**
````
a = {'일본', '중국', '한국'}  # 집합(Set) a를 생성하고 초기 요소로 '일본', '중국', '한국'을 포함

a.add('베트남')  # 집합 a에 '베트남'을 추가
a.add('중국')    # 집합에 이미 '중국'이 존재하므로 중복 요소는 추가x
a.remove('일본')  # 집합 a에서 '일본'을 제거

a.update({'홍콩', '한국', '태국'})  # 집합 a에 여러 요소('홍콩', '한국', '태국')을 한 번에 추가

print(a)  # 최종적으로 변경된 집합 a의 내용을 출력

````

**Q3 (정보처리기사 21년 1회 )**
````
class Q3:  # 클래스 Q3을 정의
li = ["seoul","kyeonggl", "lnchon", "daejeon", "daegu", "pusan"]  # 리스트 li에 여러 도시 이름을 저장
s = Q3  # 클래스 Q3의 인스턴스 s를 생성

str01 = ""  # 빈 문자열 str01을 초기화

for i in s.li:  # s의 속성 li의 각 요소 i에 대해 반복
    str01 = str01 + i[0]  # 각 도시 이름의 첫 글자를 str01에 추가

print(str01)  # 최종적으로 생성된 문자열 str01을 출력

````
